# Dynamic Weather Prediction System 

##  Project Overview
This project predicts temperature using Machine Learning 
based on humidity, wind speed, and atmospheric pressure.

##  Features
- Data preprocessing
- Visualization
- Linear Regression model
- Streamlit web app interface

##  Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit
- Matplotlib

##  How to Run
1. Install requirements
2. Run weather.py
3. Open Streamlit interface

##  Output
Real-time weather prediction system.
